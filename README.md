# Restaurant Management System 🍽️

A full-stack Restaurant Management System built with Python Flask, SQLite and Bootstrap.

## 🌐 Live Demo
[ashimdev.pythonanywhere.com](https://ashimdev.pythonanywhere.com/)

## ✨ Features
- User Registration & Login with password hashing
- Stats dashboard (today's orders, revenue, pending orders)
- Menu management (add, edit, delete, search)
- New order creation with multiple items
- Bill generation with subtotal and total
- Order completion tracking
- Responsive Bootstrap UI

## 🛠 Tech Stack
- Python + Flask
- SQLite Database
- Bootstrap 5
- Jinja2 Templates
- Flask Sessions
- Werkzeug Password Hashing

## ▶ How to Run
1. Install Flask: `pip install flask`
2. Run database setup: `python database.py`
3. Start server: `python app.py`
4. Visit: `http://127.0.0.1:5000`
